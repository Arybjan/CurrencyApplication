# Currency Application (WPF + MVVM)

## 📌 Описание

Десктопное приложение для работы с валютами с использованием API
Реализовано на WPF с применением архитектуры MVVM.

Приложение позволяет загружать актуальные курсы валют, сохранять их локально, добавлять пользовательские валюты и управлять данными.

## 🚀 Функционал

### 🔹 Основной

* Загрузка списка валют с API
* Отображение валют в таблице (DataGrid)
* Навигация между страницами
* Использование паттерна MVVM

### 🔹 Работа с данными

* Сохранение данных в JSON
* Сохранение данных в SQLite
* Переключение между JSON и SQLite
* Автоматическая загрузка данных при запуске

### 🔹 Пользовательские действия

* Добавление собственной валюты
* Удаление валют
* Обновление данных с API без потери пользовательских записей

### 🔹 Дополнительно

* Сохранение даты и времени последней сессии
* Отображение последнего запуска приложения
* Обработка ошибок (например, отсутствие интернета)


## 🧱 Архитектура

Проект построен по принципам MVVM:

* **Models** — модели данных
* **ViewModels** — бизнес-логика и команды
* **Views** — пользовательский интерфейс (XAML)
* **Services** — работа с API, JSON, SQLite и настройками


## ⚙️ Используемые технологии

* C#
* WPF (.NET)
* MVVM (CommunityToolkit.Mvvm)
* SQLite
* JSON (System.Text.Json)
* HttpClient

## ▶️ Как запустить

1. Клонировать репозиторий:

```bash
git clone https://github.com/Arybjan/CurrencyApplication.git
```

2. Открыть решение в Visual Studio 2022

3. Запустить проект (F5)

* Реализовано объединение данных API и пользовательских записей (без потери данных)
